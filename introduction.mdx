## Prerequisites

Most MAFF tools share a common set of requirements. Check the individual tool documentation for anything beyond this baseline.

- **Windows 10 or Windows 11** â€” All PowerShell-based tools target modern Windows. The ClickUp extractor and gitconfig tools also run on macOS and Linux.
- **PowerShell 5.1 or later** â€” PowerShell 7\+ is preferred for all scripts. Windows PowerShell 5.1 is supported for legacy compatibility where noted.
- **Administrator rights** â€” The majority of scripts perform system-level operations (app installation, registry changes, Active Directory queries) and require an elevated session.
- **WinGet** â€” Required for `winget-app-setup`. WinGet ships with Windows 10 1809\+ and Windows 11 via the App Installer package.
- **Active Directory module (RSAT)** â€” Required for the AD scripts inside `PowerShellScripts`. The `ActiveDirectoryUtils` module attempts to install RSAT automatically when it detects they are missing; if elevation is unavailable it falls back to a limited read-only mode.
- **Python 3.11\+** â€” Required for `clickup_task_extractor` only.

<Note>
  If you import `Modules\ActiveDirectoryUtils` without elevation, the module logs a warning and continues in limited mode rather than throwing a terminating error. Elevate your session to unlock full AD write operations.
</Note>
