## What's Included

The MAFF toolkit contains five separate tools, each solving a distinct IT automation problem:

- **winget-app-setup** — A single PowerShell one-liner that installs or updates a curated list of 10 essential Windows apps using WinGet, with support for scheduled automatic updates.
- **PowerShellScripts** — A collection of enterprise-grade scripts covering Active Directory user and computer management, date and region format standardisation, network diagnostics, and printer deployment.
- **clickup_task_extractor** — A Python CLI that pulls tasks from the ClickUp API and exports them to Markdown, HTML, or CSV, with optional Google Gemini AI-generated summaries.
- **flaui-powershell-tester** — A FlaUI-based GUI test harness that lets you run and validate PowerShell scripts against live Windows environments, including Proxmox virtual machines.
- **gitconfig** — A cross-platform Git configuration bundle featuring a terminal user interface for browsing and applying Git aliases.
